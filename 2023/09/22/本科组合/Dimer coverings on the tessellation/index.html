    Height functions

    The height function of a dimer configuration maps the vertices to , which is uniquely defined by the following rules:

    • When encircling in an black area counterclockwisely, the height ascends when the edge lies in the boundary of some dimer; descends otherwise.
    • When encircling in an white area clockwisely, the height ascends when the edge lies in the boundary of some dimer; descends otherwise.
    • Assume one given vertices is of zero height for reference.

    Such function is well-defined for any possible scheme of dimer configuration. The height function on the boundary of the region is independent of the covering.
